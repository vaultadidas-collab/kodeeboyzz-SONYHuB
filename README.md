## 🚀 Triển khai (GitHub Pages)

Không cần cài đặt gì cả — chỉ 1 file HTML.

1. Đẩy `index.html` lên repo.
2. Vào **Settings → Pages → Source: `main` / `root`**.
3. Đợi ~1 phút → web chạy tại `https://<username>.github.io/<repo>/`.

```bash
git add index.html
git commit -m "Update SonyVault Hub"
git push
```

> 💡 File ~5MB do ảnh nhúng base64. Up xong đợi GitHub build hoàn tất rồi mới mở (Ctrl+F5 để xoá cache).
